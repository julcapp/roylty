# 1. Назначение

Recipe Engine отвечает за преобразование готовой конфигурации продукта в технологический рецепт приготовления.

Он связывает цифровой выбор пользователя с физическим процессом выдачи продукта.

Recipe Engine не занимается выбором продукта, расчётом цены, оплатой или отображением интерфейса.
