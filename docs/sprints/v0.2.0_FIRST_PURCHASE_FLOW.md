# v0.2.0 — First Purchase Flow

Статус: Draft  
Проект: Soft ICE Platform / «У Тимоши»

## Цель

Создать первый рабочий пользовательский сценарий Mini App:

```text
Главный экран → Купить мороженое → Выбор сиропа → Выбор топпинга → Корзина
```

## Scope v0.2.0

В релиз входят:

1. Экран `01_HOME_SCREEN`.
2. Экран `02_PRODUCT_SCREEN`.
3. Базовая навигация между экранами.
4. Выбор сиропа из 3 вариантов.
5. Выбор топпинга из 3 вариантов.
6. Расчёт итоговой цены 130 ₽.
7. События аналитики.
8. Подготовка к корзине и оплате.

## Product assumptions

На текущем этапе автомат работает с одним вкусом дня.

```text
Вкус дня: Ванильное мягкое мороженое
Цена: 130 ₽
Выбор: 1 сироп + 1 топпинг
```

## Definition of Done

- Mini App собирается командой `npm run build`.
- Главный экран открывает экран покупки.
- На экране покупки можно выбрать сироп и топпинг.
- Итоговая цена отображается как `130 ₽`.
- События логируются через `trackEvent`.
- Код разложен по экранным модулям.
